# Fantasy Console

This will eventually be a very simple fantasy console...

## Running locally

```bash
yarn install
yarn start
```

## Build

```bash
yarn build
```

## Deploy

```bash
yarn deploy
```

## Use

Local: [http://localhost:1234/](http://localhost:1234/)
Hosted: [https://theinvader360.github.io/fantasy-console](https://theinvader360.github.io/fantasy-console)
